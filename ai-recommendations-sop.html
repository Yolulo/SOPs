What Is This?

AI Recommendations — Overview

The AI Recommendations feature uses machine learning to surface personalised merchant suggestions to customers across all storefronts. It is currently in Beta and is controlled by a single platform-wide toggle in the admin panel.

STEP 01
🧭
Navigate
Go to Merchants → AI Recommendations tab
STEP 02
✨
Locate the Setting
Find the AI Recommendations feature row
STEP 03
🔘
Enable / Disable
Toggle on or off — takes effect platform-wide immediately
ℹ️
Scope: This is a platform-wide setting. Toggling it on or off affects AI recommendations across all merchant storefronts simultaneously — it cannot be enabled for individual merchants only.
Step 01

Navigate to AI Recommendations

The setting lives inside the Merchants section alongside the Merchant and Category tabs.

Navigation Path

Step 02

The AI Recommendations Tab

The Recommendations page shows a single feature row with a toggle and a link to the Help documentation.

Fig 2.1 — AI Recommendations tab: Feature column shows the toggle; Settings column shows current state

Page Elements

AI Recommendations ✦ (Beta) — name and description of the feature
Description
"Control visibility of AI-powered recommendations across all merchant stores"
Help link
📖 Help for AI Recommendations — links to Hyperzod documentation for this feature
Settings column
Blue toggle = Enabled · Grey toggle = Disabled
Step 03

Enable or Disable AI Recommendations

One click on the toggle controls the feature platform-wide. No save button is required — the change takes effect immediately.

Toggle Reference
To enable
Click the toggle so it turns blue — AI recommendations are now active across all storefronts
To disable
Click the toggle so it turns grey — AI recommendations are hidden from all storefronts
Current state (screenshot)
Enabled — toggle is blue
Save required?
No — toggle changes save instantly
⚠️
Beta feature: AI Recommendations is currently in Beta. If you encounter unexpected behaviour after enabling, disable the toggle and raise it with the platform team before re-enabling.
💡
When to disable: Consider turning this off during major platform updates, promotional periods where you want manual control over merchant visibility, or if customer-facing recommendation quality needs review.
