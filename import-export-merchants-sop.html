Overview

Import vs Export

Both tools are accessible from the Merchants list via the Import and Export buttons in the top right corner.

IMPORT
📥
Import Merchants
Bulk-create merchants by uploading a correctly formatted CSV file
EXPORT
📤
Export Merchants
Download all merchant data as a CSV for backup or reporting

Fig 1.1 — Merchants list: Import and Export buttons in the top right corner

Import

Importing Merchants via CSV

Use this to create multiple merchants at once from a spreadsheet. The platform provides a reference CSV template — always use this.

Fig 2.1 — Merchants Import page: 3-step instructions, language selector, file upload area and import history table

Import Process — Step by Step
Step 1 — Download template
Click the blue Download link next to "Download reference CSV file" to get the official template. Do not create your own CSV from scratch
Step 2 — Populate the CSV
Fill in your merchant data following the exact column headers in the template. Any mismatch in column names or data format will cause the import to fail or upload incorrectly
Step 3 — Upload the file
Drag and drop the completed CSV into the upload area, or click Browse to select it. Select the correct language first (default: English)
Step 4 — Monitor status
The import history table below the upload area shows Date, Status, Import ID and a Download Import File link. Refresh to check progress
⚠️
Always use the reference CSV template. The platform states: "Any mismatch might result in failure or incorrect upload of your data." Double-check all columns match exactly before uploading.
Export

Exporting Merchants to CSV

Export a full CSV of all merchant records — useful for backups, reporting, or as a base for a bulk import into another platform instance.

Fig 3.1 — Merchant Export page: language selector, Export button, and export history table with Date, Export ID, Records, Language, Status and Download columns

Export Process
Step 1
Click Export on the Merchants list (top right) to open the Merchant Export page
Step 2 — Select language
Choose the language for the export (default: English)
Step 3 — Trigger export
Click the Export button. The platform queues the export job
Step 4 — Download
Once the export completes, it appears in the history table below. Click the Download link in the Download column to save the CSV file
💡
Export history is retained in the table — you can re-download previous exports by date if needed. The table shows Export ID, Number of Records, Language and Status for each job.
