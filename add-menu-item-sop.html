How to create a new product in a merchant's Catalog from inside the merchant admin panel

Getting Started

Before You Begin

Products are managed from inside the merchant's own admin panel — not the main Hyperzod admin. You must be logged in as the merchant (via Login button on the Merchants list) or have the merchant open in a separate session.

STEP 01
🧭
Navigate to Catalog
Open the merchant panel and go to Catalog → Product tab
STEP 02
📝
Product Details
Enter name, description, category, image and right-panel settings
STEP 03
💰
Pricing
Set price, compare-at price, cost price and tax settings
STEP 04
➕
Options / Add-ons
Enable multiple add-ons if the product has variants or extras
ℹ️
Access route: Main Hyperzod Admin → Merchants → find the merchant → click Login button → Catalog in the merchant sidebar.
Step 01

Navigate to the Catalog

Once inside the merchant admin panel, go to Catalog in the left sidebar and make sure you're on the Product tab.

Fig 1.1 — Merchant Catalog: Product tab selected, showing existing products with category, price and stock

Catalog Tab Overview
Product
The main product list — where you add and manage individual menu items
Category
Manage product categories within this merchant (e.g. Desserts, Sides, Curries & Mains)
Label
Create product labels (e.g. New, Popular, Spicy) that appear as badges on the storefront
Product Options
Manage reusable add-on option groups (e.g. Sauce choices, Size)
Product Scheduler
Schedule products to be available only at certain times
💡
Tip: Use Import / Export (top right) to bulk upload products via CSV. For individual items, click + Create New to open the Add Product form.
Step 02

Add Product — Details

Click "+ Create New" to open the Add Product form. The form is split into a left column (product content) and a right panel (status and settings).

Fig 2.1 — Add Product form: left column for content, right panel for status, inventory and media

Left Column — Product Content
Select Language
English (default)
SKU
Optional stock-keeping unit code — e.g. JAL for Jalebi. Leave blank if not tracking by SKU
Product Name Required
Full product name as it appears to customers — e.g. Jalebi, Basmati Rice
Description
Enticing product description. Use the ✦ Generate by AI button to auto-generate from the product name, or type manually. Rich text formatting supported
Right Panel — Status & Settings
Status Required
Set to Active for the product to be visible on the storefront immediately
Order Quantity
Toggle on to limit how many of this item a customer can order per transaction
Enable Inventory
Toggle on to track stock levels. The product will show as Out of Stock when quantity hits zero
Enable Product Instruction
Toggle on to allow customers to leave a free-text instruction for this specific item at checkout
Label
Assign a storefront badge (e.g. New, Popular) — or click Create New to make one
Tag
Add searchable tags to help customers find this product
Add Media Optional
Upload product image. Accepted: GLB, GLTF, SVG, PNG · Size: 512 × 512 px
Step 03

Product Category & Pricing

Scroll down on the Add Product form to set the product's category and pricing details.

Fig 3.1 — Add Product: Product Category selector and Pricing fields (Price, Compare-at, Cost, Margin)

Product Category
Product Category Required
Search and select the correct category (e.g. Desserts, Sides, Curries & Mains). Click Create New to add a new category on the fly
Pricing
Price Required
The selling price shown to customers — e.g. £8.00
Compare-at Price
Optional original price — displays as a crossed-out "was" price on the storefront to show a discount
Cost Price
Your internal cost — used to calculate margin and profit. Not visible to customers
Margin (%)
Auto-calculated from Price and Cost Price — read-only field
Profit
Auto-calculated — shows gross profit per unit (Price minus Cost Price)
Charge tax on this product
Check if VAT should be charged on this item specifically (only relevant if tax is not already included in prices at the merchant level)
Step 04

Options / Add-ons

The OPTIONS section at the bottom of the form allows you to link add-on groups to this product.

Options Configuration
This product has multiple Add-ons
Check this box if the product has variants or extras (e.g. sauce options, size choices, toppings). This reveals the add-on group selector
No add-ons needed
Leave unchecked for simple products with no customisation options
💡
Add-on groups are pre-configured in the Product Options tab of the Catalog. Create your option groups there first, then assign them to products here.
ℹ️
Saving: Click the blue Save button (top right) when all fields are complete. The product will appear in the Catalog list immediately and be live on the storefront if Status is set to Active.
Final Check

New Product Checklist

Confirm each item before saving.

NAME
Product name entered correctly — matches how it should appear to customers
DESCRIPTION
Description written or AI-generated — accurate and customer-friendly
CATEGORY
Correct product category assigned
PRICE
Selling price entered correctly
IMAGE
Product image uploaded at 512 × 512 px (or skipped intentionally)
STATUS
Status set to Active (or Inactive if not ready to go live)
ADD-ONS
Add-on groups assigned if product has variants or extras
SAVED
Save button clicked — product visible in Catalog list
