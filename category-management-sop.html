Getting Started

What Are Merchant Categories?

Merchant categories are the cuisine or business type labels that appear on the storefront — Burger, Pizza, Italian, etc. They help customers browse and filter merchants. This SOP covers how to view, create, edit and manage them.

Step 02

Viewing the Category List

The Category tab shows all existing merchant business categories with their sort order and active status.

Fig 2.1 — Merchants page: click the Category tab to view all business categories

Fig 2.2 — Categories list: Name, Sort Order, Status and Actions columns

Understanding the List Columns
Name
The category label shown to customers on the storefront (e.g. Burger, Pizza, Italian)
Sort Order
Controls the display order. 0 = no specific priority. Lower numbers appear first. Use the arrow icon to drag-reorder
Status
Toggle Active to show the category on the storefront, or off to hide it
Actions
✏️ Edit icon to update — 🗑️ Delete icon to remove permanently
💡
Tip: Use the Search Categories bar to quickly find a specific category in a long list.
Step 03

Adding a New Category

Click "+ Create New" (top right) to open the Add Category modal. Complete all required fields then Save.

Fig 3.1 — Add Category modal: Name, Sort Order, Status and Image upload

Add Category — Field Reference
Select Language
English (default — change only if adding a multi-language category name)
Name Required
Enter the category name as it will appear to customers — e.g. Burger, Pizza, South American
Sort Order Required
Set to 0 unless a specific display position is needed. Lower numbers appear higher in the list
Status Required
Set to Active for the category to be visible on the storefront immediately after saving
Upload Image Optional
Drag and drop or Browse to upload a category icon. Required size: 512 × 512 px. Use a clear, recognisable icon representing the cuisine or business type
⚠️
Image spec: Images must be 512 × 512 px. Uploading the wrong size may result in a stretched or incorrectly cropped icon on the storefront. Prepare the image before uploading.
💡
Naming convention: Use title case for category names (e.g. South American not south american). Keep names short — one or two words display best on the storefront category strip.
Step 04

Editing & Managing Categories

All ongoing category management is done from the Categories list using the Actions column.

Available Actions
ActionHowNotes
Edit a categoryClick the ✏️ edit icon in the Actions columnOpens the same modal as Add Category — all fields are editable
Change name or imageEdit modal → update field → SaveChanges are reflected on the storefront immediately
Reorder categoriesUpdate Sort Order value, or drag using the ↑↓ arrow iconLower number = appears first in the storefront category list
Deactivate a categoryToggle the Status switch to off (grey)Hides the category from customers without deleting it
Delete a categoryClick the 🗑️ delete iconPermanent — confirm merchants using this category are reassigned first
⚠️
Before deleting: Check that no active merchants are assigned to the category you are removing. Deleting a category that merchants are still assigned to may affect how those merchants appear on the storefront.
Quick Reference

Category Setup Checklist

Use this when adding a new category to make sure nothing is missed.

NAME
Category name entered in title case, clear and customer-friendly
IMAGE
512 × 512 px icon prepared and uploaded
SORT ORDER
Sort order set appropriately (0 for no preference, lower = higher position)
STATUS
Status set to Active so the category is visible on the storefront
VERIFY
Category appears correctly in the list and on the storefront
MERCHANT
If applicable — merchant(s) updated to use the new category
