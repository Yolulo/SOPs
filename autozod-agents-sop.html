Overview

Agents List

The Agents page lists every registered driver. View their status, activate/suspend accounts, and manage Stripe payout connections.

💡
Navigation: Autozod top nav → Agents
Agents list — Name · Email · Contact · Action (toggle, edit, delete) · Active/Suspend · Payout status
Table Columns
Name
Profile photo and full name.
Email
Registered email — used for driver app login.
Contact
Phone number including country code.
Action
Green toggle — enable/disable account. ✎ — edit agent. 🗑️ — permanently delete agent.
Active/Suspend
Active — can receive orders. Suspended — blocked from orders but account kept.
Payout
Connected ✓ — Stripe linked. Connect button (purple) — ready to onboard. Grey toggle — Stripe not initiated.
Agents list default view showing Active/Suspend status and Payout connection states per agent.
⚠️
Deleting an agent is permanent. Use Suspended to temporarily block a driver instead.
Adding

Add a New Agent

Create a driver account so they can log into the Autozod app and receive order assignments.

💡
Navigation: Agents → Add button (top right, blue)
Add Agent modal — fill all required fields and click ADD.
Form Fields
First Name Required
Agent's first name.
Last Name Required
Agent's last name.
Email Required
Unique email address — used for driver app login.
Contact Number Required
Phone with country code. Use the flag selector for the correct prefix.
Password Required
Initial password. Share securely with the driver — ask them to change on first login.
Profile Photo Optional
PNG, JPG or JPEG. Appears on the dashboard map and agents list. Click Choose File.
Clicking Choose File opens the system file picker. Supported: PNG, JPG, JPEG.
📱
After adding: Share the agent's email and password so they can log into the Autozod driver app. Once logged in they appear as Available on the dashboard.
Searching

Search for an Agent

Quickly find a specific driver using the search bar on the Agents page.

How to Search
Search box
Type any part of the agent's name into the Search agents box.
Search button
Click the blue Search button to filter the list.
Clear search
Clear the box and click Search again to return to the full list.
Search agents — type a name or partial name and click Search to filter results in real time.
Editing

Edit an Agent

Update an agent's name, email, contact number, password or profile photo at any time.

Opening the Edit Modal
How to open
In the Action column, click the ✎ pencil icon for the agent you want to update.
Edit Agent modal — all fields pre-filled. Created At is read-only.
Editable Fields
First / Last Name
Update the agent's display name.
Email
Changing email changes the driver app login credentials.
Contact Number
Update phone number. Country flag selector available.
Password / Confirm Password
Leave blank to keep existing password. Enter both fields to reset it.
Profile Photo
Current photo shown. Click Choose File to replace.
Created At
Read-only. Original account creation date.
EDIT button
Click to save all changes. Modal closes and list updates immediately.
