Step 1

Accessing Autozod

Autozod is accessed directly from the Hyperzod admin sidebar — no separate login required.

The AUTOZOD widget is pinned to the bottom-left of the Hyperzod admin sidebar. Click it to open Autozod in a new tab.
How to Access
From the sidebar
Click the AUTOZOD widget at the bottom-left of the Hyperzod admin sidebar (black panel, triangle logo, "Automatic order assignment" label). Opens in a new tab.
Top navigation
Once inside Autozod: Dashboard · Agents · Task history · Payouts · Settings · Webhooks. Store switcher and account name top-right.
💡
Autozod is a separate system — changes here affect live driver dispatch and order assignment in real time.
Step 2

The Live Dashboard

The Dashboard is the real-time dispatch view — live map, all active tasks, and every agent's current status.

Autozod Dashboard — Tasks panel (left), live map (centre), Agent panel (right)
Tasks Panel (Left)
Active
Orders currently out for delivery. Count in brackets.
Assigned
Orders assigned to a driver but not yet picked up.
Unassigned
Orders awaiting a driver — needs attention if count is non-zero.
Scheduled
Future-dated orders queued for dispatch.
Agent Panel (Right)
Inactive
Agents offline / not accepting orders.
Available
Agents online and ready. Shown in green with current order count.
Busy
Agents currently on a delivery.
Map pin
Green dot = last known agent location. Updates in real time.
🗺️
No Tasks showing? Normal when no active orders. Agents can still appear as Available on the right panel.
